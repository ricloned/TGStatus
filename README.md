## Автоматическая смена телеграм статуса и эмодзи статуса

![Пример](res/expl.png)

# Что это?
Скрипт автоматически получает список Ваших публичных подарков, выбирает рандомный и ставит его в эмодзи статус (премиум). Также раз в n минут изменяет статус "О себе"

# Как использовать?
1. Заполнить config.json своими данными
2. Запустить main.py
3. Создать свою сессию телеграм
4. Оставить скрипт работать
